# Smart-Expense-Tracker

## Steps to run the project
1. `git clone https://github.com/rashmikantprajapti21-wq/Smart-Expense-Tracker.git`
2. Now, it will have two Folders Frontend and Backend.
3. Open the Frontend in Visual Studio Code.
4. Run `npm install`
5. Now, open the backend in InteliJ Idea
6. Run the backend now
7. Now, open Visual Code and run `npm run dev`
